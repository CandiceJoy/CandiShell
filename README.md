## Install
```
wget -O ~/bootstrap.sh https://cdn.jsdelivr.net/gh/CandiceJoy/CandiShell/bootstrap.sh; chmod 700 ~/bootstrap.sh; ~/bootstrap.sh
```

## Update
```
wget -O ~/.zshrc https://cdn.jsdelivr.net/gh/CandiceJoy/CandiShell/.zshrc
wget -O ~/.tmux.conf https://cdn.jsdelivr.net/gh/CandiceJoy/CandiShell/.tmux.conf
```
### Or
```
wget -O ~/update-shell.sh https://cdn.jsdelivr.net/gh/CandiceJoy/CandiShell/update-shell.sh; chmod 700 ~/update-shell.sh; ~/update-shell.sh
```
